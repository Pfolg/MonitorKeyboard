# 运行

双击运行

# 缺陷
+ 不可自定义
+ 无托盘，只能在任务管理器中关闭
+ 只支持Windows

*可以使用[源码](src\v3.cpp)自行编译*

# 优势
+ 实时监控
+ 全局监控
+ 简单易用
+ 无需安装
+ 体积极小
+ 内存占用低
+ 限制单实例
